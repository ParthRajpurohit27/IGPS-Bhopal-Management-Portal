# **App Name**: ScholastiGate

## Core Features:

- Teacher Attendance Log: Simple one-tap interface for teachers to mark presence, capturing name, employee ID, and high-precision timestamps.
- Secure Multi-Role Access: Secure login system for Teachers, Owners, and Admin using 10-digit mobile numbers or emails, handled via a secure database authentication system.
- Admin Control Center: Master interface for account creation, password resets, and managing the 15-20 teacher profiles and administrative credentials.
- Intelligent Status Tool: An automated tool that uses logic to analyze missing records and designate teachers as absent or on holiday based on the school schedule.
- Academic Holiday Manager: Administrative calendar tool to set holidays, ensuring teachers are correctly credited and not marked absent on school-wide breaks.
- Historical Record Editor: Enables Admin to manually update or override attendance status for any current or past dates as needed.
- Payroll Summary Engine: Report generator that filters data by teacher name and custom date ranges to calculate total presence, absences, and holidays for salary processing.

## Style Guidelines:

- The palette is based on Academic Integrity. Primary: Professional Blue (#284F7F), Background: Pristine Slate (#F6F8F9), and Accent: Bright Indigo (#6B6BCE).
- A pairing of 'Inter' for functional UI labels and 'Literata' for historical records and payroll headers to evoke a scholarly, formal feel.
- Clean, line-art style icons representing clocks, calendars, and personnel records to maintain professional clarity.
- Dashboard-first layout featuring a simplified single-button view for teachers and a high-density data grid for admin management.
- Status indicators use subtle color-fades when transitioning between 'Present', 'Absent', and 'Holiday' states.
